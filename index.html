plain javascript popup plugin


NOTI is a notifier plugin written in plain JavaScript with no dependecies needed. Just load it and fire it up. Itâ€™s fully custimizable via plugin options and CSS.



install github





install

        
          <!-- include noti stylesheet -->
          

          <link rel="stylesheet" type="text/css" href="noti.min.css">
          

          

          <!-- include noti script -->
          

          <script src="noti.min.js"></script>





context

NOTI provides four context types: default, success, warning and error




position

NOTI provides six positions: top-left, top-center, top-right, bottom-left, bottom-center and bottom-right







api

        
          var myNoti = noti({

              content: 'My super awesome content!!!',

              autoOpen: false

          });
          

          

          myNoti.open();
          

          

          myNoti.close();


options

        
          noti({

              type: 'default', // string

              content: '', // string

              position: 'top-right', // string

              autoOpen: true, // boolean

              autoOpenDelay: 10, // number

              autoClose: false, // boolean

              autoCloseDelay: 1500 // number

          });
